windows in my crib

tldr; window creation in zig made me sad, so i went with rust instead

a very good place to start

let’s start from the very beginning, and put a smile on julie andrews’ face. i set out to make this a rust app, then promptly got whispers of zig. heard about ghostty, got interested, cloned, reviewed, and got sad. ghostty, a terminal emulator written in zig (zig resources: book, karl seguin, ziglings) by mitch hashimoto (he flies a cirrus, by the way!!!) is currently avaialable for linux (wayland, i think), and mac. i looked around to see how window creation was handled and found it was handled natively.

this is where we are

i have made the executive decision to stick with tauri + svelte. i would have gone with react, but i wanted something much lighter; bonus for being declarative.
